# Data Schema (v1)

All game content is loaded from JSON files in `/data`.

- `data/map/sol.map.json`: node and route graph.
- `data/cards/base.cards.json`: starter deck cards.
- `data/resources/resources.json`: resource types and starting values.
- `data/rules/*.json`: economy, build costs, combat, victory constants.
- `data/scenarios/standard_1v1.json`: players, starting fleets, round limit, hand size.
